# 2023-beginner-challenge-1

### 📄 요구사항

1. 상위메뉴 클릭 시, 하위메뉴가 슬라이드로 애니메이션 효과

2. 하위메뉴 클릭 시, 선택된 하위 메뉴만 활성화

3. 뒤로가기 기능 지원

4. 새로고침 시 해당 페이지의 토글 메뉴 오픈 활성화

### 📝 코드 리뷰 방식

➡️ 개별 작업 후 모여 코드 공유 및 구현이 이뤄지지 않은 코드에 대한 리뷰. <br>

뒤로가기 기능 구현과, 슬라이드 애니메이션 구현이 완료되었고 가독성이 좋음.
