### 코드 리뷰 방식
구현 중 고민이 되는 부분이 있으면 서로 의논해보았습니다. 리뷰할 때는 서로 코드에 대해 설명하고, 서로 궁금한 로직을 질문하고 답변하는 방식으로 진행했습니다.
